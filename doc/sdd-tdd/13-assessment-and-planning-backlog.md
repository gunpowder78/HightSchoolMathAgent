# 学情评估与个性化补习计划工具 Backlog

## 1. 目的

本文件将学情评估与个性化补习计划工具拆分为可开发的 issue，供后续按 `SDD/TDD` 顺序实施。

## 2. Issue 列表

### `ISSUE-APT-01` 学情问卷与输入模型

- 对应需求：
  - `APT-FR-01`
  - `APT-FR-02`
  - `APT-FR-03`
- 目标：
  - 定义基础档案、数学经历和心理压力的结构化输入。
- 交付物：
  - 输入 schema；
  - 前端表单；
  - 持久化结构。

### `ISSUE-APT-02` 防御底盘评分器

- 对应需求：
  - `APT-FR-04`
- 目标：
  - 根据基础题、第一问和可操作分数池计算底盘。
- 交付物：
  - 评分规则；
  - 单元测试；
  - 结构化输出。

### `ISSUE-APT-03` 认知风格识别器

- 对应需求：
  - `APT-FR-05`
- 目标：
  - 标注视觉型、机械执行型、代数脆弱型等风格标签。

### `ISSUE-APT-04` 模块能力矩阵生成器

- 对应需求：
  - `APT-FR-06`
- 目标：
  - 生成按模块拆分的能力矩阵与等级。

### `ISSUE-APT-05` 风险识别引擎

- 对应需求：
  - `APT-FR-07`
- 目标：
  - 识别心理风险、黑名单污染、时间预算和生态冲突风险。

### `ISSUE-APT-06` 动态防御策略生成器

- 对应需求：
  - `APT-FR-08`
- 目标：
  - 生成白名单/黑名单、第一问覆盖和多选策略建议。

### `ISSUE-APT-07` 双主线计划生成器

- 对应需求：
  - `APT-FR-09`
  - `APT-FR-10`
- 目标：
  - 生成主线 A、主线 B、日计划、周计划和阶段目标。

### `ISSUE-APT-08` 熔断修复协议库

- 对应需求：
  - `APT-FR-11`
  - `APT-FR-13`
- 目标：
  - 对高频 Bug 提供结构化修复协议。

### `ISSUE-APT-09` 三题切片复评器

- 对应需求：
  - `APT-FR-12`
- 目标：
  - 支持视觉复现题、机械执行题、费曼逆向讲解题三类复评。

### `ISSUE-APT-10` 家长执行面板

- 对应需求：
  - `APT-FR-14`
- 目标：
  - 输出今日重点、禁止项、鼓励话术和风险提示。

### `ISSUE-APT-11` 计划版本管理

- 对应需求：
  - `APT-NFR-02`
  - `APT-NFR-05`
- 目标：
  - 保存计划版本、变更原因和生效状态。

### `ISSUE-APT-12` 测试数据集与评测脚本

- 对应需求：
  - `APT-NFR-04`
- 目标：
  - 建立评估样本、风险样本和计划回归数据集。
