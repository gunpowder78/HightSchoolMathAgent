## 3. 当前页面能力

`Gradio` 页面“学情评估”页签已支持以下完整链路：

1. 学情评估；
2. 计划生成；
3. 三题复评与熔断修复；
4. 自动调参与版本对比；
5. 家长面板与打卡；
6. 趋势与周报；
7. 快照导出；
8. 阈值告警与图表；
9. 批量导出与运维巡检。

## 4. 测试状态

- 单元测试：覆盖评估、计划、复评修复、家长面板、报告、运维模块。
- 集成测试：覆盖 `SQLite` 的评估、计划、复评、修复、打卡、阈值、巡检查询链路。
- 当前基线结果：
  - `python -m pytest` 通过（全绿）。

## 5. 已知限制

- 图表目前采用轻量 HTML 表格形式，而非完整图表库。
- 批量导出默认以全量学生为目标，尚未提供复杂筛选条件。
- 阈值策略仍是规则驱动，后续可升级为学习型阈值推荐。

## 6. 下一迭代建议

- 引入更强图表组件并支持时间窗口过滤；
- 增加批量导出的分组筛选与失败重试；
- 对高风险学生增加“主动告警推送”渠道（如邮件/IM）。
